
## Getting Started

To get started with the Resume Maker on your device, follow these steps:

1. **Clone the Repository**:
```bash
  git clone https://github.com/yourusername/resume-maker.git
```
2. **Install Dependencies**:
```bash
  cd resume-maker
  npm install
```
3. **Run the Application**:
```bash
  npm run dev
```
